# Student Task Manager

A simple Full Stack Task Management Web Application built using the MERN Stack.

## Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB Atlas

## Features
- Add Tasks
- Delete Tasks
- Mark Tasks as Completed
- REST API Integration
- MongoDB Cloud Database

## Project Structure
client/ -> Frontend
server/ -> Backend
